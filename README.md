# Currency-Converter

💱 Currency Converter Web App
📌 Project Overview

  This project is a Currency Converter Web Application that allows users to convert one currency into another using real-time exchange rates fetched from a public API.
  The application provides a clean and modern UI with light and dark mode support, ensuring a better user experience across devices.

🎯 Purpose of the Project

  The goal of this project is to demonstrate:
  
    API integration using fetch
    
    Dynamic UI updates using JavaScript
    
    Use of Array Higher Order Functions (HOFs)
    
    Responsive and interactive web design
    
🌐 API Used

This project uses the following public API:

Currency API
  👉 https://latest.currency-api.pages.dev/v1/currencies/${COUNTRY_CODE}.json
  
🔹 Features of API:
  Provides real-time exchange rates
  Supports multiple currencies
  Free and easy to use
  
🚀 Features
  ✅ Core Features
      Convert currency values in real-time
      Select "From" and "To" currencies
      Swap currencies with a single click
      Dynamic exchange rate display
      
  ⚙️ Interactive Features
      🔍 Input validation (handles empty/invalid input)
      🔄 Currency swap functionality
      🎨 Light / Dark mode toggle
      ⚡ Loading state while fetching data
      📱 Fully responsive design
      
  🧠 JavaScript Concepts Used
      fetch() for API calls
      map() → rendering currency options
      filter() → searching currencies
      sort() → organizing currency lists
      Event handling & DOM manipulation
      
🛠️ Technologies Used
  HTML5 → Structure
  CSS3 → Styling & responsiveness
  JavaScript (ES6) → Logic & API handling
